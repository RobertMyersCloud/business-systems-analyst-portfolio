# Inventory Analysis Workbook

## Business Problem
Leadership needs a clear view of inventory value, low-stock risk, overstock risk, aging inventory, and warehouse distribution.

## Tools Used
- Microsoft Excel
- Excel tables
- Formulas
- SUMIFS / COUNTIFS
- Conditional formatting
- Dashboard design
- Charts
- Data dictionary

## Skills Demonstrated
- Cleaned and structured inventory data
- Calculated inventory value
- Identified reorder risk
- Identified overstock items
- Built warehouse and category summaries
- Created an executive-style dashboard
- Documented business findings and data definitions

## Key Outputs
- Inventory analysis workbook
- Dashboard preview
- Warehouse inventory summary
- Category inventory summary
- Low-stock and overstock indicators
- Aging inventory indicators

## Business Value
This workbook helps leadership identify stockout risk, excess inventory, and inventory value across warehouses so they can make better purchasing and planning decisions.

## Portfolio Context
This project is part of a Business Systems Analyst portfolio focused on Excel, SQL, Power BI, ERP processes, IAM, and GRC controls.
